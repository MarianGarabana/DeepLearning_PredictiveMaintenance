# Demo Data Format

Generated by `notebooks/04_evaluation.ipynb` after model training.

Each file is a JSON array:
```json
[
  { "cycle": 1, "sensors": [float x 21], "rul": 125.0 },
  { "cycle": 2, "sensors": [float x 21], "rul": 125.0 },
  ...
]
```

Files:
- `engine_healthy.json`   — stable capped RUL 125, no visible degradation
- `engine_degrading.json` — RUL drops 79→0 over 80 cycles  ← USE FOR DEMO
- `engine_critical.json`  — RUL already <50, near failure

These are pre-recorded so the demo NEVER depends on live model inference.
The Simulate button reads this file cycle-by-cycle at 1s intervals.
